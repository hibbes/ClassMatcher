Schiller-Klassen-Mixer auf Windows installieren

Eine einzelne EXE-Datei. Keine Installation in C:\Program Files nötig — du legst sie ab, wo's logisch ist, und klickst doppelt.

Aktuelle Version __VERSION__
Schritt 1

App herunterladen

Klick auf den Knopf. Edge, Chrome oder Firefox funktionieren alle.

Wichtig vorab: Die App läuft komplett auf deinem PC. Es werden keine Daten ins Internet übertragen. Die einzigen Internet-Zugriffe sind ein gelegentlicher Check auf neuere Versionen und der Download dieser neueren Version, falls du zustimmst.
Schiller-Klassen-Mixer für Windows herunterladen (.exe)
Wenn nach dem Klick gar nichts passiert oder eine Warnung kommt: Manche Browser blockieren den Download stumm, weil die App nicht im Microsoft-Store ist. So kommst du trotzdem an die Datei:
  1. Rechtsklick auf den Knopf oben.
  2. Im Menü „Link speichern unter …" oder „Ziel speichern unter …" wählen.
  3. Es erscheint ggf. ein Warn-Dialog „Diese Datei kann dein Gerät beschädigen". Klick „Trotzdem behalten" (Chrome/Edge) oder „Beibehalten" (Firefox).
  4. Die Datei landet jetzt im Downloads-Ordner.
Der Browser warnt nur, weil die App keine kostenpflichtige Microsoft-Entwickler-Signatur hat. Sie ist trotzdem sicher, sie kommt direkt vom Schiller-Gymnasium.
Schritt 2

Die EXE bereitlegen

  1. Öffne den Datei-Explorer (Symbol mit gelbem Ordner unten in der Taskleiste, oder Windows-Taste + E).
  2. Geh links in der Seitenleiste auf Downloads. Dort liegt die Datei Schiller-Klassen-Mixer-v__VERSION__.exe (etwa 12 MB). Das ist schon die fertige App, da muss nichts entpackt werden.
Tipp: Zieh die EXE per Drag-and-Drop an einen Ort mit Schreibrechten, etwa auf den Desktop oder in einen eigenen Ordner. Dann findest du sie leicht wieder und das spätere Auto-Update kann sie dort selbst austauschen. In einen geschützten Ordner wie C:\Program Files gehört sie NICHT, sie braucht keine klassische Installation und ist eine einzige in sich geschlossene Datei.
Schritt 3

App zum ersten Mal starten

  1. Doppelklick auf die EXE-Datei.

Wenn alles glatt läuft, öffnet sich nach ein, zwei Sekunden ein Browser-Fenster mit der Adresse http://localhost:5001.

Sehr wahrscheinlich kommt aber beim ersten Start eine Sicherheits-Warnung von Windows, weil die App nicht über den Microsoft-Store ausgeliefert wird:

Wenn Windows die App blockiert

Fall A„Der Computer wurde durch Windows geschützt"

Ein blaues Fenster mit dem Titel „Der Computer wurde durch Windows geschützt" erscheint und nennt SmartScreen als Quelle.

  1. Im Fenster siehst du den Text „Microsoft Defender SmartScreen hat den Start einer unbekannten App verhindert." Darunter ein kleiner Link „Weitere Informationen" oder „Mehr Informationen". Klick darauf.
  2. Das Fenster wird länger und zeigt den App-Namen sowie einen neuen Knopf „Trotzdem ausführen" unten rechts. Klick diesen Knopf.
  3. Die App startet jetzt. Windows merkt sich diese Entscheidung, ab dem nächsten Mal genügt ein normaler Doppelklick.

Fall BAntivirus hat die Datei in Quarantäne verschoben

Bei manchen Antivirus-Programmen (Avast, Kaspersky, Norton …) verschwindet die EXE direkt nach dem Entpacken. In dem Fall hat das Antivirus sie als „verdächtig" markiert und in einen Quarantäne-Bereich verschoben.

  1. Öffne deine Antivirus-Software (Symbol in der Taskleiste unten rechts, eventuell unter dem kleinen Pfeil „verborgene Symbole anzeigen").
  2. Such den Bereich „Quarantäne" oder „Verlauf".
  3. Such den Eintrag Schiller-Klassen-Mixer.exe und klick „Wiederherstellen" bzw. „Vertrauen / Whitelist".
  4. Zurück in den entpackten Ordner, EXE wieder starten.

Wenn deine Antivirus-Software es zulässt, kannst du die Datei zusätzlich als Ausnahme hinzufügen, damit sie bei zukünftigen Updates nicht wieder blockiert wird.

Was passiert beim Start?

Sobald die App läuft, öffnet sich automatisch ein Browser-Tab mit http://localhost:5001. Das sieht aus wie eine Website, ist aber keine: der PC spricht hier nur mit sich selbst. Zusätzlich erscheint ein schwarzes Konsolenfenster — das gehört zur App, ignorieren, nicht schließen.

Zum Beenden schließt du den Browser-Tab. Die App beendet sich nach etwa 30 Sekunden von selbst — kein Hintergrund-Prozess bleibt liegen.

Automatische Updates ab dieser Version: Beim Start prüft die App selbständig auf neuere Versionen. Falls eine da ist, erscheint oben ein orange-gelbes Banner mit „Jetzt herunterladen". Auf Windows wird die neue EXE im Hintergrund geholt und an genau die Stelle gelegt, an der deine aktuelle EXE liegt. Nächster Start → neue Version aktiv. Du musst also nur diese erste Installation manuell durchziehen.

Was tun, wenn nichts klappt?

Schreib Marek, am besten mit:

  • Einem Screenshot der Fehlermeldung. Taste Druck oder PrtScn, dann in eine Mail einfügen.
  • Welche Windows-Version du hast: Start → Einstellungen → System → Info.
  • An welchem Punkt der Anleitung du hängst.

So lässt sich am schnellsten weiterhelfen.
